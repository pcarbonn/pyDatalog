# Support & community

Please report bugs [here](https://www.google.com/url?q=https%3A%2F%2Fgithub.com%2Fpcarbonn%2FpyDatalog%2Fissues%2Fnew&sa=D&sntz=1&usg=AOvVaw0tfwNlUE8MTHOdO6a8y5Iv), on GitHub.

Please post questions on the use of pyDatalog [on StackOverflow](http://www.google.com/url?q=http%3A%2F%2Fstackoverflow.com%2Fquestions%2Fask&sa=D&sntz=1&usg=AOvVaw14sLAZt1ej7HkhNbxaln1T) (make sure to add the [pydatalog tag](https://stackoverflow.com/questions/tagged/pydatalog) at the bottom of the form). I'll respond as quickly as possible.

Other questions / comments can be posted [on the pyDatalog Google Groups](https://groups.google.com/forum/#!forum/pydatalog). Feel free to subscribe to it for announcements.

Please note that it is recommended to [upgrade to the latest release](installation.md).
