pyDatalog uses a JIT lua compiler.  Unfortunately, official binaries are not available on the web, so it needs to be compiled on the target machine.

Installation instructions (DRAFT, to be further tested)

* install a C compiler, e.g. Windows SDK (recommended ?), Visual C++, minGW or CygWin
* download and unpack lupa
* download and unpack luajit in a subdirectory of lupa, and build it
* build and install lupa
* finally, download pyDatalog
